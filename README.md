
# TUI-do

> A beautiful, terminal-native Kanban board that manages your tasks in Git-friendly Markdown. 

**Note: This project is currently in active development for FOSS Hack.**

## What is TUI-do?

TUI-do brings project management directly to where developers already live: the terminal. It is a blazing-fast, visually rich Terminal User Interface (TUI) that reads and writes to a simple `.tui-do.md` file stored right inside your project repository. 

Say goodbye to context switching and proprietary cloud databases. With TUI-do, your tasks are version-controlled alongside your source code.

## Planned Features

* **Frictionless Flow:** Open your board instantly with a single command (`tuido`).
* **Git & Markdown Native:** Local, offline-first, and completely file-based.
* **Modern Terminal UI:** Mouse support, Vim motions, and clean aesthetics.
* **Branch Awareness:** Highlights tasks based on your current Git branch.

## Tech Stack

* **Language:** Python
* **UI Framework:** [Textual](https://textual.textualize.io/)
